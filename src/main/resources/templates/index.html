Kubernetes, AWS Cognito & Lambda

Digital on Us exercise

Develop a microservice with the following characteristics

  • It must be done with spring boot
  • It must be deployed and running in Kubernetes.
  • It displays one HTML page with a login button. After the user logs in, it shows an HTML page with a greeting message including the logged in user name
  • Login is handled by AWS Cognito.
  • Greeting message is retrieved from a AWS Lambda function, input: <username>, output: Hello <username>
